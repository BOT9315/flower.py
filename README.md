# 🌸 flower.py

A Python script that draws beautiful flower patterns using the Turtle graphics library.

---

## 📸 Preview

> _Run the script to see the flower rendered live in a graphics window._

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your machine
- The `turtle` module (included in Python's standard library — no extra install needed)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/BOT9315/flower.py.git
   cd flower.py
   ```

2. **Run the script:**

   ```bash
   python flower.py
   ```

   A graphics window will open and draw the flower automatically.

---

## 🧠 How It Works

The script uses Python's built-in `turtle` module to draw flower petals by:

- Moving the turtle cursor in arcs to form individual petals
- Rotating the turtle by a fixed angle between each petal
- Repeating the petal shape to complete the full flower

## 🛠️ Customization

You can tweak the following variables inside `flower.py` to change the appearance:

| Variable      | Description                        |
|---------------|------------------------------------|
| `num_petals`  | Number of petals in the flower     |
| `petal_size`  | Size/length of each petal          |
| `color`       | Fill color of the petals           |
| `speed`       | Drawing speed of the turtle        |
